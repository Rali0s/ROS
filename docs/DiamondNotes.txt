Progressive Disclosure Behavior
Do not show all apps at once.
Each doctrine node should expose only a small app tray.
Preferred behavior:
Default state: show the diamond and maybe one selected category tray.
Hover/focus category: subtly reveal that category’s top 3 apps.
Click category: expand that category tray with more apps.
Click app: open the existing module.
Click center: open the Operator Memory Cockpit.
This avoids a cluttered app grid while still making the system navigable.
App Tray Design

small icon
app name

no large descriptions unless expanded

Beta install
Install the desktop app, then create your encrypted workspace.

ROS Beta is a native desktop app. Core use does not require cloud sync. Local model workflows use the model runtime on your machine when available, and manual capture remains active when it is not.

macOS

Download the DMG, move ROS into Applications, launch it, and create or unlock the local workspace.

Windows

Run the installer package, complete setup, then start ROS from the desktop shortcut or Start menu.

Linux

Use the AppImage or distro package. Appliance images are planned separately from desktop beta builds.

First launch checklist
  • Create a master passphrase for the encrypted workspace.
  • Create your first project workspace.
  • Capture one note, command, file reference, or model output.
  • Search active project memory.
  • Export a recovery bundle before trusting ROS with daily work.

Trust posture
Encrypted at rest. Decrypted for the session. Local by default.

ROS Beta is built around a local desktop trust boundary. Unlocking decrypts the workspace into memory for the active session. The passphrase is not stored, and core workflows do not require a cloud account.

Model workflow

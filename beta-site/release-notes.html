Version 1.2.0-beta.1
Production-beta polish across trust, notes, and Nostr Lounge.
  • Native-vault migration and desktop trust posture remain the default release path.
  • Backup export, dry-run restore validation, support bundle export, and beta diagnostics are in place.
  • Nostr Lounge now includes identity export, wallet mirroring, per-relay diagnostics, and a read-first profile flow.
  • Wallet Vault now renders revealed Nostr keys in a readable expanded layout.
